# Battleship Game (in Kotlin!)

## Rules

Two player turn based game. 

Each player has an Ocean Grid and a Target Grid. Each player
hides their respective grids from the other.

There are five types of ships:

- carrier (5)
- battleship (4)
- cruiser (3)
- submarine (3)
- destroyer (2)
